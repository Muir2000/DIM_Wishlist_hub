## 한눈에 보기

- **무엇**: 데스티니 가디언즈 유저가 무기/퍽을 클릭만으로 골라 **DIM 호환 위시리스트**를 만들고 `.txt` 로
  내보내거나, 남의 위시리스트를 가져와 점수화하는 웹 서비스.
- **상태**: 백엔드(FastAPI)+컴파일러+프론트(React) 동작·검증 완료. Docker Compose 배포(기본 127.0.0.1,
  `WEB_BIND` LAN, NPM+Let's Encrypt 외부 HTTPS — [07](07-외부공개-HTTPS배포.md)). 샘플 데이터로 즉시 구동,
  매니페스트 적재 시 전체 무기/퍽(2059종).
- **주요 기능**: 헤더 검색 + 컨텍스트 인지 필터/DIM식 텍스트 쿼리, 변형·시즌(복각/홀로포일) 처리,
  비주얼 퍽 그리드(툴팁·실시간 스탯·퍽별 점수 배지), 위시리스트 가져오기/내보내기,
  **위시리스트 기반·퍽 롤 중심 점수화**(스코프 블렌드·열 비중·동적 만점·태그별 점수·최적 롤),
  **내 창고 정리**(검색/정렬/다중퍽/다중선택 DIM id 복사/중복 탐지), **Bungie 로그인 멀티유저**(사용자별 저장).
- **언어/타겟**: **한국어 기본 + 영어 토글**(매니페스트 ko 공식 용어), 한국 커뮤니티.
- **품질**: 단위 테스트 74개(컴파일러 30 + 쿼리 21 + 점수 23). 보안 검증 21/21(인증 이전 표면; 멀티유저
  항목은 보강 예정 — [05](05-보안검증.md)).
