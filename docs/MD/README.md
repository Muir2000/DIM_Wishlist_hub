## 한눈에 보기

- **무엇**: 데스티니 가디언즈 유저가 무기/퍼크를 클릭만으로 골라 **DIM 호환 위시리스트**를 만들고 `.txt` 로
  내보내거나, 남의 위시리스트를 가져와 점수화하는 웹 서비스.
- **상태**: 백엔드(FastAPI)+컴파일러+프론트(React) 동작·검증 완료. Docker Compose 배포(포트 127.0.0.1).
  샘플 데이터로 즉시 구동, 매니페스트 적재 시 전체 무기/퍼크(2059종).
- **주요 기능**: 헤더 검색 + 컨텍스트 인지 필터/DIM식 텍스트 쿼리, 변형·시즌(복각/홀로포일) 처리,
  비주얼 퍼크 그리드(툴팁·실시간 스탯), 위시리스트 가져오기/내보내기, 위시리스트 기반·퍽롤 중심 점수화.
- **언어/타겟**: 한국어 우선 UI(매니페스트 ko 공식 용어), 한국 커뮤니티.
- **품질**: 단위 테스트 74개(컴파일러 30 + 쿼리 21 + 점수 23), 보안 검증 21/21 통과.
